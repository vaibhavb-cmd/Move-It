# Move-It
Responsive moving service website built using HTML, CSS, and Bootstrap. Features modern UI components like navbar, hero section, carousel, and responsive layout.

# 🚚 Move It - Moving Service Website

Move It is a responsive frontend website for a fictional moving service company.
It is designed using Bootstrap components to create a clean, modern, and user-friendly interface.

---

## 🚀 Features

* Responsive navigation bar with dropdown menu
* Hero section with call-to-action buttons
* Feature section highlighting services
* Image carousel for visual presentation
* Clean and structured footer
* Fully responsive design (mobile, tablet, desktop)

---

## 🛠️ Tech Stack

* HTML5
* CSS3
* Bootstrap 5

---

## 📸 Project Overview

This project demonstrates:

* Use of Bootstrap components (Navbar, Grid, Cards, Carousel)
* Responsive layout design
* Clean UI structure for real-world business websites
* Reusable and scalable frontend components
